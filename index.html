What is EduSahara?

EduSahara™ is a simple yet very powerful tool that lets aspiring students reach their goal of excelling in Mathematics.
The strength of EduSahara™ is the ability of the software to generate infinite questions. Questions are generated using a Randomizer. So, every time the question is generated, it shows up with different data.
EduSahara™ covers topics in Mathematics from VI to X class of CBSE, ICSE and A.P. state board syllabus.
Student can practice with unending generation of questions from each chapter to get complete understanding of the concepts.
Teachers have the flexibility to create assignments by selectively picking questions from different chapters.Read More

    2. Click on Practise
    3. The EduSahara login page is displayed.
    4. Now, click on the browser home (red icon on the top right). On some browsers, it will be a 3 vertical dots (ellipsis) icon. Select the 'Add to Home Screen' menu item. A popup is thrown asking for the name of the icon. Enter 'EduSahara' and click add. The link will be added as a desktop icon. From now onwards, the user can click on this icon to access EduSahara.
    5. In the login page, login using the credentials provided. If an account already exists, use those credentials.
    6. The application is best rendered in the horizontal view of the phone (landscape mode).
